# grocery-billing-receipt-generator

This is a Java-based OOP project that simulates a grocery billing system.

## 🚀 Features
- Add multiple items
- Automatic total calculation
- 5% tax calculation
- Clean receipt generation
- Input validation

## 🛠 Technologies Used
- Java
- OOP Concepts (Encapsulation, Aggregation)
- ArrayList
- StringBuilder

## ▶️ How to Run
1. Compile:
   javac Main.java

2. Run:
   java Main

## 📸 Sample Output
------ Grocery Bill ------
Item    Price   Qty   Total
Rice    50      2     100
Milk    30      3     90

Subtotal: 190
Tax (5%): 9.5
Grand Total: 199.5
